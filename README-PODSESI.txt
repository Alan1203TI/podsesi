PODSESI Player - versão tablet sem categorias

Arquivos principais:
- index.html: tela dos alunos
- admin.html: painel administrativo
- app.js: carrega episódios e busca
- admin.js: envia capa, áudio e cadastra episódio no Firebase
- firebase-config.js: configuração do Firebase

Senha padrão do painel admin:
podsesi2026

Nesta versão as categorias foram removidas.
Os episódios aparecem na tela dos alunos em ordem de cadastro, do mais recente para o mais antigo.
A tela dos alunos possui apenas o campo de busca.

Regras:
- Use o arquivo REGRAS-FIRESTORE.txt no Firestore
- Use o arquivo REGRAS-STORAGE.txt no Storage
